Markdown Cheat Sheet

Your quick reference guide for Markdown syntax.

Headings

Use hashes (#) to create headings. More hashes indicate a smaller heading level.

# Heading 1
## Heading 2
### Heading 3

Emphasis

Use asterisks (*) or underscores (_) to add emphasis to your text.

*Italic*
**Bold**

Lists

Create bullet lists using dashes (-), pluses (+), or asterisks (*). Use indentation for nested items.

- Item 1
- Item 2
  - Sub-item

Links

Format links with square brackets for the text and parentheses for the URL.

[Link Text](http://example.com)

Images

Images are similar to links but with an exclamation mark at the start.

![Alt Text](http://example.com/image.jpg)

Code Blocks

Use triple backticks (```) to create code blocks. You can specify the language for syntax highlighting.

```python
def hello_world():
    print("Hello, world!")
```

Blockquotes

Use the greater than symbol (>) to create blockquotes. Useful for highlighting quotes or special content.

> This is a blockquote.

Tables

Create tables with pipes (|) and hyphens (-). Use pipes to separate columns and hyphens to create headers.

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
